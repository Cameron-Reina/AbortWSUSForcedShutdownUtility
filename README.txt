        A utility to prevent unwanted Windows Server Update Services forced shutdowns
     ----------------------------------------------------------------------------------

                                    To use the utility:

               Simply double-click the AbortWSUSForcedShutdownUtility.vbs file.

The script will run in the background, continuously monitoring for any scheduled shutdown requests.
  This utility will automatically abort a shutdown, ensuring your system remains up and running.
